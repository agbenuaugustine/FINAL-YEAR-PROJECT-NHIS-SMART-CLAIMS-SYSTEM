Key Features

  • 🎯 Context-Aware Actions: Different buttons shown based on hospital status
  • ✨ Modern Design: Clean, Tailwind-inspired styling with hover effects
  • 📱 Responsive: Adapts to mobile screens with icon-only mode
  • 🔒 Safe Operations: Multi-step confirmation for destructive actions
  • ⚡ Real-time Feedback: Loading states and success/error notifications
